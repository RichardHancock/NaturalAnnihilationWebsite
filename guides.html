Install Guide

Windows

There are two available Windows downloads. The standalone executable once launched will start the game without any install process. While the installer will install the game to its own folder, create desktop shortcuts and add it to the start menu.

Locate the file you downloaded, which will be named similarly to this:

Launch the file, if you downloaded the standalone the game will now launch, but if you downloaded the installer the install window will appear.

"Read" through the GameMaker licence agreement and decide whether you agree to the terms, by clicking "I Agree" or "Cancel". You need to agree to use this program.

The next part of the installer lets you select which optional components you want to include in the installations. These options are:

  • Start Menu Shortcuts
  • Desktop Shortcut

Next you have the option of setting a custom install directory. For standard users leave this as default.

The installer will now start installing the game, this can take a few minutes.

The game has now been successfully installed and can be auto launched if you tick the "Start Natural Annihilation" option.
